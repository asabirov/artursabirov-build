ElasticSearch Server

В книге рассказывается об установке базы, плагинов и вариантах запуска. Описывается REST API, через который осуществляется всё взаимодействие с базой. Об индексации, геоиндексах, фильтрах и разновидностей поиска. О том как использовать фасеты, управлять нодами и кластерами.

Также автор не обходит стороной и практическое применение встроенных инструментов: подсветка найденного, автокомплит, поэтапная подгрузка результатов, кэширование.

Книга идеальна для первого знакомства с ElasticSearch.
